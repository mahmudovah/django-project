# Birinchi Django app

## Qo`llanma

- Birinchi o`zimizga kerakli papka yaratib olamiz
- virtual muhit yaratamiz
- virtual muhitni aktivlashtiramiz
- kerakli paketlarni o`rnatamiz

## Buyruqlar

Django topshiriq yaratish
```bash
django-admin startproject core .
```

Django app yaratish
```bash
python manage.py startapp main
```

Migratsiya yaratish
```bash
python manage.py makemigrations
```

Migratsiyalarni amalga oshirish
```bash
python manage.py migrate
```

Serverni ishga tushirish
```bash
python manage.py runserver
```
